# Установка
* Клонируем или скачиваем последний релиз
* Копируем файл *env-example* в *.env*, при необходимости меняем настройки

## Linux

Запуск
```bash
./start
```

Остановка
```bash
./stop
```

Пересобрать контейнеры
```bash
./rebuild
```

## Windows

Запуск
* start.bat

Остановка
* stop.bat

Пересобрать контейнеры
* rebuild.bat

## Системные требования
Свободного места - 2 GB
Docker
Docker compose
